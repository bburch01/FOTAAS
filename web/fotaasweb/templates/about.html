About FOTAAS

FOTAAS is a non-commercial sofware system that is intended to be provided to
prospective employers for demonstration and code review purposes.

FOTAAS demonstrates proficiency in the following areas:

  • Golang
  • Golang Concurrency/Parallelism
  • Golang Code Generation
  • Golang Web Application Development
  • Protobuf
  • gRPC
  • Microservices
  • Docker
  • Kubernetes
  • Cloud Deployment
